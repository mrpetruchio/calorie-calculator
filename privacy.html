Политика конфиденциальности

Этот документ описывает, какие данные NutriWorld собирает и как они используются.

1. Общие положения

NutriWorld — сервис для расчёта калорий, поиска блюд и ведения дневника питания. Используя сайт, пользователь соглашается с условиями настоящей Политики конфиденциальности.

Настоящая Политика составлена для MVP-версии проекта и может быть обновлена при развитии сервиса.

2. Какие данные мы собираем

Сайт может обрабатывать следующие данные:

  • имя пользователя;
  • email;
  • данные авторизации через Supabase;
  • записи дневника питания: блюда, граммы, калории, белки, жиры, углеводы, дата;
  • локальные настройки сайта: язык, цель калорий, профиль расчёта.

Сайт не запрашивает паспортные данные, адрес проживания, номер телефона, точную геолокацию или медицинские диагнозы.

3. Для чего используются данные

Данные используются только для работы функций сайта:

  • создание аккаунта;
  • вход пользователя;
  • сохранение дневника питания;
  • отображение истории питания;
  • расчёт дневной нормы калорий;
  • улучшение пользовательского опыта.

4. Где хранятся данные

Авторизация и облачные данные хранятся в Supabase. Часть настроек может храниться локально в браузере пользователя через localStorage.

Пароли пользователей не хранятся в коде сайта. Авторизация обрабатывается Supabase.

5. Доступ к данным

Пользователь имеет доступ только к своим записям дневника. Для защиты данных используются правила доступа Supabase RLS.

6. Удаление данных

Пользователь может очищать записи дневника питания на сайте. В дальнейшем будет добавлена функция полного удаления аккаунта и связанных данных.

7. Ограничение ответственности

NutriWorld не является медицинским сервисом. Расчёты калорий и БЖУ носят справочный характер. При медицинских ограничениях или заболеваниях необходимо консультироваться со специалистом.

8. Контакты

По вопросам обработки данных можно обращаться к владельцу проекта NutriWorld.
